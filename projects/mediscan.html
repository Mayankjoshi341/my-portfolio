🩺 MediScan – Health Risk Guidance System

🚩 Problem

Traditional symptom checkers often provide vague or incomplete results, lacking actionable insights and real-world guidance.

💡 Solution

Built an ML-powered system that predicts diseases based on user symptoms and provides detailed guidance including precautions, severity, and recommendations.

🔥 Key Features

  • Top 3 disease predictions with probability
  • Severity level (1–5 scale)
  • Precautions and home remedies
  • Location-based hospital recommendations
  • Interactive map integration
  • Email report generation

🧠 Tech Stack

Python, Scikit-learn, Pandas, NumPy, Streamlit, JavaScript

📊 Dataset

Synthetic healthcare dataset (IBM)

📈 Results

Achieved ~99% accuracy (due to synthetic dataset) and built a complete end-to-end deployed system.

⚠️ Note

Model performance may vary on real-world data. Future work includes training on real clinical datasets.
